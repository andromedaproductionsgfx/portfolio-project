# Cafe POS System

A web-based Point of Sale (POS) system designed for cafes and restaurants.

## Features
- Order and billing system
- Product and category management
- Customer handling
- Deals and combo offers
- Multi-branch support
- Real-time interface
- Supabase integration

## Tech Stack
- HTML
- CSS
- JavaScript
- Supabase

## Purpose
This project demonstrates a complete front-end POS system with real-world business use case.
